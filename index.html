PowiainaNum.js

A large number library reaching {10,9e15,1,1,1,2} in Bower's operator notation.

For smaller numbers, use other libraries such as:

  • break_infinity.js by Patashu - e9e15
  • Decimal.js by MikeMcl - e9e15
  • logarithmica_numerus_lite.js by Aarex Tiaokhiao - e1.8e308
  • confractus_numerus.js by Aarex Tiaokhiao - ee9e15
  • magna_numerus.js by Aarex Tiaokhiao - ?
  • break_eternity.js by Patashu - 10^^1.8e308
  • OmegaNum.js by Naruyoko - 10{1000}9e15
  • ExpantaNum.js by Naruyoko - 10{1000}9e15

How it works

This library works by storing the sign, layer, and an array.
The sign is either 1 or -1, positive or negative respectively.
Array holds small part of the magnitude; It can be respented in TypeScript: 

typeof PowiainaNum.array = {
    arrow: number,
    expans: number,
    megota: number,
    repeat: number,
}[]

In PowiainaNum.js 0.1.x version, it was 
[number, ...([number|"x", number, number|"x", number])[]]
